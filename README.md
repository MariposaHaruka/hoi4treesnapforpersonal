[아래 언급되지 않은 내용은 이쪽 참고](https://github.com/cpntodd/hoi4treesnap)

## 추가 사항

- 한국어 렌더링 지원
- focus의 titlebar_style 지원
- shared_focus와 joint_focus가 연결되어 있는 경우 같이 출력됩니다.

## 주의 사항

- **포크의 포크를 한 프로젝트이니 완성품은 별도 배포는 안합니다.** 필요한 경우 코드 참조는 하시는 건 자유입니다.
- 리눅스 환경이 아니라서, 해당 환경에서 사용할 경우 별도 수정이 필요합니다.
- 출력되는 이미지의 중점명이 한국어가 기본 설정으로 되어있기에, **한국어가 아닌 다른 언어 사용시 언어 설정 꼭 하신 뒤에 이미지를 생성하시기 바랍니다.**
- shared_focus와 joint_focus 코드를 보신 분들은 알겠지만, offset 때문에 국가별로 위치가 천차만별인 경우가 다양합니다. 이 부분은 직접 수작업으로 수정해주시길 바랍니다.
- (원본부터 당연한 이야기지만) allow_branch 옵션이 적용되지 않습니다. **이미지 스냅샷을 생성할 경우 별도의 폴더에 중점 파일을 복사해서 스냅샷용으로 수정하는 걸 권장합니다.**
